作品集 / Portfolio


此作品集包含我本人独立制作及与他人共同制作的游戏、手机应用、手绘作品等，均为课程作业和学术研究项目，语言为英语，部分使用网络开放资源。请勿以任何形式抄袭或盗用。


This portfolio is a collection of my game design, application design and drawing works. All of them are course and research projects in the English language, some of which use open assets online. Plagiarism in all forms PROHIBITED.

音嬉 / Musices

“音嬉”是一款作曲教学手机应用，提供和弦拼接、乐器演奏、五线谱和midi乐谱三种作曲方式，同时还提供乐队功能，使用户可以协作演奏或作曲。

Musices is a mobile application for music composition, providing three ways of composing with chord splice, musical instruments, score and midi notations. Band function enables users to work collaboratively on composing.

3D拼图游戏 / 3D Puzzle Game

此游戏使用Unity开发，通过旋转视角拼成规定的3D图形，旨在提升少儿的空间推理能力。

This is a 3D puzzle game demo with three perspectives, which aims to improve the spatial reasoning ability of young kids. The game was developed with Unity.

白日梦 / Daydream

“白日梦”是一款使用橙光游戏制作工具制作的互动叙事游戏，致力于使玩家意识到性别问题不是改变某一个个体的举动，更重要的是打破社会成见的固化思维。

Daydream is a constructivist interactive narrative game developed with Chengguang Game Maker, aiming to raise awareness about how generalized and fossilized social expectations affect women.

意外 / Surprise

此作品是在以“Surprise”为题的1.5小时游戏开发挑战中完成的作品，使用Unity开发，旨在讲述每个人在路上遇到的困难和挣扎，以及坚持走下去终将看到的曙光。

The game was created in a 1.5-hour game jam with Unity, which uses the theme of “surprise” to allude to all the fears, distractions and negative feelings on our way, as well as to an unexpected destination where the light shines ahead.

梦游宋都 / Wandering in Song

“梦游宋都”是一款音乐跑酷游戏，通过躲避随机障碍和碰撞“宫商角徵羽”音符演奏乐曲，游览夜景。

“Wandering in Song” is a music parkour game with Song-style cultural landscape, in which players avoid random obstacles and collect music notes to play Chinese ancient songs.

回绿 / H&MxGreen

“回绿”是以H&M公司为利益相关者、为宣传其环保品牌所做的手机应用，旨在为旧衣回收提供便利，提升用户的环保意识。

H&MxGreen is a mobile application designed to boost users’ consumption of H&M’s Conscious clothing line and encourage users to recycle old clothes and develop environmental awareness.

板绘及炭笔画 / Digital Drawings & Charcoal Drawings

此手绘合集为课程作业练习，参照网络原图，使用炭笔和数位板绘制。

The drawings (unoriginal) use online pictures as references, drawn with a digital tablet and charcoals.
